#  Victory Vault – Sports Statistics Platform

**Victory Vault** is a web-based sports statistics platform that provides structured insights into **Cricket**.  
It offers dynamic fixtures, match results, player performance analytics, and league statistics — all powered by an efficient backend and clean visual dashboards.

---

##  Features

###  Fixtures & Results  
- Displays upcoming fixtures and completed match results.  
- Data updates dynamically from the backend using aggregation pipelines.  

###  Smart Data Management  
- Efficiently stores and retrieves data using MongoDB Aggregations.  
- Includes advanced projections to filter and display only relevant stats for Cricket.  

###  Data Visualization  
- Interactive **bar** and **line** charts to visualize player and team performance.  
- Helps users compare stats like runs, wickets, averages, and more.  

###  Cricket Data Module  
- Currently supports **Cricket** data.  
- Includes player profiles, match data, and team statistics.
